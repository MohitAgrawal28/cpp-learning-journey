## 🛠️ How to Compile and Run

### Using g++ (GCC/MinGW)
```bash

g++ filename.cpp -o filename
.\filename.exe
```

### Using Visual Studio or VS Code
- Open the file and use the built-in build task (Ctrl + Shift + B)
- Run the compiled executable
